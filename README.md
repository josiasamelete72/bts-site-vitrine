## Description du projet
Ce projet consiste à créer un **site web de présentation et de vente de services**.  
Le site est simple, professionnel et responsive, adapté pour un portfolio ou une vitrine de services.  

Le site contient :  
- Une **page d’accueil** (`index.html`) présentant les services principaux.  
- Une **page complémentaire** (`page2.html`) présentant des services supplémentaires.  
- Un **formulaire de contact fonctionnel** permettant d’envoyer des messages directement aux étudiants via Formspree.

---

## Objectifs
- Créer un site clair et professionnel pour présenter des services.  
- Apprendre à structurer un site web avec HTML et CSS.  
- Implémenter un formulaire de contact fonctionnel sans serveur.  

---

## Technologies utilisées
- **HTML** – Structure des pages  
- **CSS3** – Design et mise en page
